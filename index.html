About This Project

This website is created as part of the 645 course assignment. The project demonstrates the fundamentals of web development including HTML5, CSS3, and responsive design using Bootstrap framework. The website is hosted on Amazon Web Services (AWS) using both S3 static website hosting and EC2 instance deployment.

As a student pursuing knowledge in cloud computing and web technologies, this assignment provides hands-on experience with real-world deployment scenarios. The integration of front-end development with cloud infrastructure showcases the modern approach to building and deploying web applications.
